This plugin provides full featured archive management widget with multiple instance and configurable options.

== Description ==
This is a full featured multiple instance supported archive manage widget with configurable options to control parameters like archive type, format,limit etc features in your sidebar.This plugin displays the archive using wordpress wp_get_archives function.

**Features**
 
* Multiple instance of the widget can be created
* Easy to use widget control panel
* Multiple archive parameter configuring options
* and many more

== Installation ==

1. Download "archive-manage-widget" plugin
2. Open /wp-content/plugins folder at your web server
3. Upload the folder "archive-manage-widget" there
4. Goto Plugins page in your wordpress back office
5. Activate "Archive Manage Widget" plugin
6. Go to Settings >> Archive Manage Widget
6. Go to Appearance >> Widgets
7. Drag your plugin to the right sidebar to activate your plugin
8. Note: This plugin will require one time free registration

== How to use it ==

* Go to Widgets in your admin panel > drag the Archive Manage Widget in the right sidebar
* Click in right side of the widget to open the configuration panel
* In the title section enter the title you want to be appeared in the front end sidebar at the top of your archive listing
* Type is the type of archive you want to be displayed 
* Limit is the number of archive to get
* Format are explaied as follows
* html format displays archive list within the <li></li> tag
* option format displays the select box of the selected type
* link format displays data within <link> tag
* In custom format archive are listed within before and after strings
* Before and After format is used to put the archive within some string that is placed in before and after
* Show Post Counts shows the total post in the archive
